Class WordNetMapper

source code

this class provides various methods to map lexkey | offset | ilidef to lexkey | offset | ilidef. each method contains a docstring that explains the full process.

In terms of efficiency, the main idea behind this setup is that once a particular method is called, for example offset_to_offset between two wordnet versions, the mapping will stay in memory. Hence, when the same method is called for the same wordnet versions, the process will be very quick.

load_bin_if_needed(self, current_path_bin, new_path_bin, attribute)

source code 

method checks if the new_path_bin is different from currently in memory. if so new_path_bin is load into memory, else nothing is done. If the combination of source and target wordnet is not available, IOError is raised.

Parameters:
  • current_path_bin (str) - full path to bin that is currently in memory (with first usage this variable will be an empty string
  • new_path_bin (str) - full path to new path of the bin which might be loaded.
  • attribute (str) - mapping_offset_to_offset | mapping_offset_to_lexkey | mapping_lexkey_to_offset

map_offset_to_lexkey(self, offset, lemma, source_wn_version)

source code 

method tries to return the lexkey of an offset. the following strategy is applied: (1) map offset to possible sensekeys (2) check for direct lemma match in possible sensekeys (3) check if only one sensekey (4) pick lexkey with lowest Levenshtein distance to lemma Exception ValueError is raised if no lexkey is found.

>>> my_mapper = WordNetMapper()
>>> my_mapper.map_offset_to_lexkey("05262185","moustache","30")
'moustache%1:08:00::'
>>> my_mapper.map_offset_to_lexkey("03413428","gambling hell","30")
'gambling_hell%1:06:00::'
Parameters:
  • offset (str) - 8 character offset with trailing zeros (for example 02102663)
  • lemma (str) - the lemma corresponding to the offset (for example '1000')
  • source_wn_version (str) - source wn_version (for example '21')
Returns: str
lexkey if found, else Exception ValueError is raised.

map_offset_to_offset(self, offset, source_wn_version, target_wn_version, output_format='highest')

source code  
>>> my_parser = WordNetMapper()
>>> my_parser.map_offset_to_offset("00020846", "21", "30")
('00021939', 'n')

method tries to map offset to offset across versions of wordnet ValueError is raised if no mapping available.

Parameters:
  • offset (str) - 8 character offset with trailing zeros (for example 00003469)
  • source_wn_version (str) - source wn_version (for example '2.1')
  • target_wn_version (str) - source wn_version (for example '3.0')
  • output_format (str) - if 'highest': str is returned of (offset,pos) with highest confidence. if 'all', a dict is returned mapping the (offset,pos) -> confidence (float)
Returns: tuple | dict
if param output_format == 'highest': str is returned of (offset,pos) with highest confidence. if param output_format == 'all', a dict is returned mapping the (offset,pos) -> confidence (float). ValueError is raised if no mapping available.

overlap(self, source_wn_version, target_wn_version)

source code 

this method determines the percentage of offset that can be succesfully mapped between two wordnet version

>>> my_parser = WordNetMapper()
>>> succes_rate,missed_mappings = my_parser.overlap("30","171")
>>> succes_rate
0.9489831562625135
Parameters:
  • source_wn_version (str) - source wn_version (for example '21')
  • target_wn_version (str) - source wn_version (for example '30')
Returns:
(succes_rate,missed_mappings). succes_rate, float between 0 and 1 indicating the percentage that was succesfully mapped. missed_mappings is a list of offsets that were unable to be mapped
